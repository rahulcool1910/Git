show all branches:

	git branch

	git checkout <branch name>

	git checkout -b <branch name>

	git ls-tree -r <branch name>

Delete branch:
	git branch -d <branch name>

	git tag <tag name> (soft tag)

	git tag -a <tag name> <annotated tag>
